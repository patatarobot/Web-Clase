Hardware

¿Qué es el hardware?

Como hemos comentado anteriormente el hardware es el conjunto de elementos fisicon que componen un ordenado una una rede de estos como son los monitores teclado raton, los componentes dentro de la torre y elementos perifericos como las impresoras. Es la parte que uedes ver y toocar de un ordenador o un conjunto de estos.



Componentes:

  • CPU: Que se ocupa de procesar datos y controlar todos los procesos.

  • Memoria RAM: Es la memoria de “trabajo” del sistema, la que se utiliza en cada momento para la ejecución de aplicaciones.

  • Unidades de almacenamiento: Dispositivos destinados al almacenamiento de los datos del sistema o de usuario.

  • Tarjeta Madre: El eje central, esqueleto o soporte en el cual se encuentran conectados los componentes electrónicos del ordenador.

  • Fuente de alimentación: La que se ocupa de suministrar electricidad a los distintos circuitos del aparato electrónico al que se conecta el PC.

  • Memoria ROM (Read Only Memory): Memoria de lectura, es decir, los datos grabados pueden ser leídos pero no pueden ser modificados por el usuario.

  • Memoria Cache: Se podría definir como un sistema de almacenamiento de alta velocidad.

  • Disco Duro: Mediante un sistema de grabación magnética, se convierte en un dispositivo de almacenamiento de archivos digitales.

  • Monitor, teclado, ratón.


    • tipos de hardware:

  • Procesamiento: encargado de ejecutar las instrucciones enviadas por el propio software del dispositivo.

  • Almacenamiento: partes físicas del ordenador que se encargan de almacenar información, como archivos o programas.

  • Periféricos de entrada: son los que permiten al usuario ingresar información desde el exterior(teclado, ratón, escáner, ...).

  • Periféricos de salida: que transmiten la información desde el ordenador hacia al usuario u otra computadora o red(manitor, impresora, altavoces, ...).

  • Periféricos de entrada y salida: partes físicas que permiten la entrada y salida de la información(microfonos, usb, discos duros, ...)
